# ostrom-frontend

A crud project.

live demo
* https://ostrom-frontend.herokuapp.com

## Requirements:
- Yarn
- NodeJs
- Git
- nestjs

## Dependencies (libraries):
- ReactJs
- Typescript
- Redux

## Installation:
- clone the repository
- run `yarn`
- update config/config.json
- run `yarn start`
